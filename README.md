# KiwoomOpenAPI

키움증권 OpenAPI Wrapper 패키지.  
Windows COM 기반으로 파이썬 32비트로만 동작.  
KiwoomTraderV2 프로젝트 패키지의 내부 모듈을 독립적인 패키지로 분리하는 작업 완료.
