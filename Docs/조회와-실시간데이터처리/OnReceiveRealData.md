[OnReceiveRealData()이벤트]

OnReceiveRealData(
BSTR sCode,        // 종목코드
BSTR sRealType,    // 실시간타입
BSTR sRealData    // 실시간 데이터 전문 (사용불가)
)

실시간시세 데이터가 수신될때마다 종목단위로 발생됩니다.
SetRealReg()함수로 등록한 실시간 데이터도 이 이벤트로 전달됩니다.
GetCommRealData()함수를 이용해서 수신된 데이터를 얻을수 있습니다.
